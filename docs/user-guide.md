# BookFlow User Guide

Start the program and type commands at the `>` prompt.

1. Add books with `book add`.
2. Register readers with `reader add`.
3. Create a loan with `loan create`.
4. Return a book with `loan return <bookId>`.
5. Use `search <text>` and `filter ...` to find books.
6. Use `stats` and `report` to inspect the library state.
